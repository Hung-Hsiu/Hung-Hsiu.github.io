# 一個「#」的標題

## 兩個「#」的標題

### 三個「#」的標題


前面加上「* 」是清單
* Red
* Green
* Blue

**前後兩個「*」是粗體**

*前面一個「 *」，沒有空格是斜體

***前後三個「 * 」是粗體+斜體***

~~前後兩個「~」是刪除線~~

超連結的語法是：用「[]」將文字包著，然後在後面加上「()」，並在括號內輸入網址
圖片插入方式是超連結的語法，並在「[]」前面加上「!」

回到我的[GitHub](https://hunghsiuwang.github.io/)


>前面一個「>」是縮排一個Tab
>>前面兩個「>」是縮排兩個Tab
